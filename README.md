# FastVLM-Agent Benchmark

## Overview
This project implements a prototype benchmark for evaluating multimodal AI agents.

## Features
- Image + question input
- Latency measurement
- Basic accuracy evaluation

## How to Run
pip install -r requirements.txt
python main.py

## Metrics
- Latency
- Accuracy

## Goal
To study trade-offs between speed and reasoning in multimodal systems.
